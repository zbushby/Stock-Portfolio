# Strategy

---

## Current Focus Period

Q1 2026 — Learning to code Python by building a Stock Portfolio Optimiser.

## Strategic Priorities

1. **Learn Python fundamentals** — variables, types, functions, control flow, data structures, OOP, error handling
2. **Learn developer tooling** — terminal, venv, pip, git, project structure, linting
3. **Build a working Efficient Frontier optimiser** — from fetching data to plotting results
4. **Learn intermediate/advanced concepts through the project** — multithreading, type hints, testing, CLI design, file I/O, decorators, generators
5. **Write production-quality code** — clean structure, proper packaging, documentation

## What Success Looks Like

- Can write Python confidently from scratch without hand-holding
- Understands core CS concepts: data structures, algorithms, concurrency
- Has a complete, working stock portfolio optimiser that:
  - Accepts a list of tickers
  - Fetches historical data
  - Calculates optimal portfolio weights on the efficient frontier
  - Uses multithreading/multiprocessing for performance
  - Visualises results with plots
  - Runs from the command line
- Understands *why* each piece of code works, not just *that* it works

## Key Decisions

- **Python first** — it's the best language for this domain (finance, data, ML)
- **Real project, not exercises** — every concept is learned in context
- **Incremental build** — start from hello world, end with a threaded optimiser
- **Explain everything** — no concept is too basic to explain when it first appears
