# Project Info

---

## Project Overview

**Stock Portfolio Optimiser** — a command-line tool that:
1. Accepts a list of stock tickers (e.g. AAPL, GOOGL, MSFT)
2. Fetches historical price data from Yahoo Finance
3. Calculates the **efficient frontier** using Monte Carlo simulation & scipy optimisation
4. Finds the **optimal portfolio weights** (max Sharpe ratio, min variance)
5. Visualises results with matplotlib/plotly
6. Uses **multithreading/multiprocessing** for performance

## Tech Stack

- **Python 3.12** — language
- **yfinance** — stock data
- **pandas** — data manipulation
- **numpy** — numerical computing
- **scipy** — optimisation
- **matplotlib/plotly** — visualisation
- **click** — CLI interface
- **pytest** — testing

## Key Context

This is a **learn-to-code project**. Every line of code is written as a learning exercise. Claude acts as a coding tutor — teaching concepts through the real project, never through detached theory. The curriculum has 32 lessons across 7 phases, from "Hello World" to a threaded, production-quality optimiser.
